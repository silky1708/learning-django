## learning-django

#### Creating a new project 
```py
django-admin startproject <project name>
```

#### Starting the Django server  
```py 
python manage.py runserver 
```    


#### Database  

```py 
python manage.py makemigrations   
python manage.py migrate 
```   


#### admin credentials  

```py 
python manage.py createsuperuser 
```
